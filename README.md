# Discord Music Bot  🇵🇱

A Discord bot featuring a YouTube music queue system and live PLN exchange rates.

## Features
- **Advanced Music Queue:** Supports play, skip, pause, resume, and auto-play.
- **Finance:** Live exchange rates (PLN to USD/TRY).

## Installation
1. Install [FFmpeg](https://ffmpeg.org/).
2. `pip install -r requirements.txt`
3. Add your tokens to a `.env` file.
4. Run `python main.py`.
